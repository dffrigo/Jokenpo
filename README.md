## Installation

Windows based
- JavaScript (Vanilla)
- Node.js v16.14.0.

Requirements:
```sh
npm install prompt-sync
```

## Usage

The game Rock, Scissors and Paper, also known as Jokenpo, is a children's game that basically works with luck.
But, strategy and prediction don't be left out.
